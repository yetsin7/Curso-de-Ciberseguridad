### Módulo 03: Linux para Ciberseguridad

Dominio de la terminal, administración y seguridad en sistemas Linux.

#### Archivos:
- `01_comandos_basicos.sh`: Script con comandos esenciales de navegación y gestión.
- `02_gestion_permisos.md`: Guía avanzada sobre permisos (rwx, SUID, SGID, Sticky Bit).
- `03_harderning_ssh.sh`: Script para asegurar el servicio SSH en el servidor.
- `04_analisis_logs.sh`: Herramienta simple para buscar intentos de intrusión en auth.log.
- `05_ejercicios_linux.txt`: Taller de administración de sistemas.
