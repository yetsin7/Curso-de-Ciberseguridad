# Módulo 03 — Linux para Ciberseguridad (Explicado Fácil 🐧)

Linux es el sistema operativo favorito de los hackers (de los buenos y de los malos). ¿Por qué? Porque te da el control total. Es como si en Windows fueras un pasajero y en Linux fueras el mecánico que puede desarmar el motor mientras el coche corre.

---

## 🧐 ¿Qué aprenderás aquí?

Aprenderemos a movernos por la terminal sin miedo. Olvida los iconos y el ratón, aquí mandamos con palabras.

| Concepto | Analogía de la vida real |
| :--- | :--- |
| **Terminal / Consola** | Un traductor que le dice a la computadora exactamente qué hacer. |
| **Directorio (Folder)** | Una habitación de tu casa. |
| **Ruta (Path)** | Las indicaciones para llegar a una habitación (ej: Casa -> Cocina -> Cajón). |
| **Root (Superusuario)** | El dueño de la casa que tiene las llaves de TODAS las puertas. |
| **Kernel** | El director de la orquesta que decide qué instrumento toca y cuándo. |

## 🔑 Los Permisos: rwx
Imagina un archivo como una caja.
- **r (Read/Lectura):** Puedes mirar qué hay dentro de la caja.
- **w (Write/Escritura):** Puedes meter cosas nuevas o sacar lo que hay en la caja.
- **x (Execute/Ejecución):** Si la caja tiene un juguete a cuerda, puedes darle cuerda para que funcione.

¡Si no tienes la 'x', el juguete no se mueve!
