# Módulo 03: Linux para Ciberseguridad

Linux es el sistema operativo preferido por los profesionales de seguridad. Aprenderás a dominar la terminal y las herramientas del sistema.

## Temario

1.  **¿Por qué Linux?**: Distribuciones enfocadas en seguridad (Kali, Parrot, BlackArch).
2.  **La Terminal (Bash)**:
    - Navegación y gestión de archivos.
    - Permisos de archivos (chmod, chown).
    - Gestión de procesos.
3.  **Administración del Sistema**:
    - Gestión de usuarios y grupos.
    - Instalación de software (apt, pacman, binarios).
    - Redes en Linux (ip, ifconfig, netstat).
4.  **Bash Scripting Básico**: Automatización de tareas simples.
5.  **Seguridad en Linux**: El archivo `/etc/shadow`, `/etc/passwd` y privilegios de sudo.

## Laboratorio Práctico
- Configuración de una máquina virtual con Kali Linux o Parrot OS.
- Creación de un script en Bash para escanear puertos abiertos de forma básica.
