# Módulo 06: Seguridad en Aplicaciones Web (OWASP Top 10)

La web es uno de los vectores de ataque más comunes. Aprenderás a identificar y mitigar las vulnerabilidades más críticas según OWASP.

## Temario

1.  **¿Qué es OWASP?**: El proyecto abierto de seguridad de aplicaciones web.
2.  **A01: Control de Acceso Roto**: Bypass de autenticación y escalada de privilegios.
3.  **A03: Inyección**:
    - **SQL Injection (SQLi)**: Manual y con herramientas.
    - **Command Injection**: Ejecución de comandos en el servidor.
4.  **A02: Fallos Criptográficos**: Uso de algoritmos débiles o falta de HTTPS.
5.  **Cross-Site Scripting (XSS)**: Reflejado, Almacenado y Basado en DOM.
6.  **Broken Object Level Authorization (BOLA)**: Vulnerabilidades en APIs.
7.  **Herramientas de Auditoría Web**: Burp Suite (el estándar de la industria) y OWASP ZAP.

## Laboratorio Práctico
- Uso de Burp Suite para interceptar y modificar peticiones HTTP.
- Explotación controlada de SQL Injection en un entorno de laboratorio (DVWA).
