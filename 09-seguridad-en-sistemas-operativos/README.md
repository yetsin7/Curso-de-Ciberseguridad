# Módulo 09: Seguridad en Sistemas Operativos (Hardening)

No todo es atacar. Aprenderás a configurar sistemas operativos para que sean resistentes a intrusiones.

## Temario

1.  **Principios de Hardening**: "Mínimo privilegio necesario" y "Reducción de la superficie de ataque".
2.  **Seguridad en Windows**:
    - Directivas de Grupo (GPO).
    - Windows Defender y AppLocker.
    - Auditoría de eventos y logs.
3.  **Seguridad en Linux**:
    - Configuración de Firewalls (iptables / nftables / ufw).
    - SSH Hardening (desactivar login de root, uso de llaves).
    - SELinux y AppArmor.
4.  **Gestión de Parches**: Estrategias de actualización en entornos empresariales.
5.  **Detección de Intrusos (HIDS)**: Instalación de Wazuh o OSSEC.

## Laboratorio Práctico
- Aplicación de un checklist de seguridad (CIS Benchmark) a un servidor Linux recién instalado.
- Configuración de una política de contraseñas robusta en Windows.
