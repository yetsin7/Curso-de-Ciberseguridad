# Módulo 09: Seguridad en Sistemas Operativos (Hardening)

Configuración robusta para minimizar la superficie de ataque.

#### Archivos:
- `01_hardening_principles.txt`: Menor privilegio y defensa en profundidad.
- `02_linux_audit.sh`: Script para auditar configuraciones críticas de seguridad.
- `03_windows_gpo.md`: Directivas de grupo esenciales para seguridad.
- `04_fail2ban_config.json`: Ejemplo de configuración para protección contra fuerza bruta.
- `05_iptables_rules.sh`: Script con reglas de firewall recomendadas.
