## Para que sirve este curso

Este curso tiene como objetivo proporcionar una base sólida y avanzada en seguridad informática. Aprenderás desde los fundamentos de redes hasta técnicas avanzadas de pentesting, análisis de malware y respuesta a incidentes.

## Modulos del curso

2. `02-fundamentos-de-redes-y-protocolos`

16. `16-proyecto-final-ciberseguridad`
17. `17-preparacion-certificaciones-profesionales`

## Resultado esperado

Al terminar el curso, deberías ser capaz de realizar auditorías de seguridad, entender y mitigar ataques comunes, y estar preparado para obtener certificaciones profesionales en el área.

## Que estudiar despues

Si quieres profundizar en áreas específicas relacionadas, te recomendamos:

1. Redes Avanzadas.
2. Programación en Python para Seguridad.
3. Administración de Sistemas Linux Avanzada.
