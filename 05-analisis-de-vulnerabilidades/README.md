# Módulo 05: Análisis de Vulnerabilidades

Identificar debilidades antes que los atacantes. Aprenderás a usar escáneres profesionales y a interpretar sus resultados.

## Temario

1.  **Ciclo de Gestión de Vulnerabilidades**: Identificación, clasificación, priorización y remediación.
2.  **Bases de Datos de Vulnerabilidades**: CVE (Common Vulnerabilities and Exposures), CVSS (puntuación de severidad).
3.  **Escaneo de Red con Nmap**:
    - Tipos de escaneo (Syn, Connect, UDP).
    - Detección de servicios y versiones.
    - Nmap Scripting Engine (NSE).
4.  **Escáneres Automáticos**:
    - Nessus (Instalación y uso básico).
    - OpenVAS / Greenbone.
5.  **Falsos Positivos**: Cómo verificar manualmente una vulnerabilidad reportada.

## Laboratorio Práctico
- Escaneo completo de una red local usando Nmap.
- Análisis de una máquina vulnerable con Nessus Essentials.
