# Módulo 04: Criptografía y Esteganografía

La ciencia de ocultar y proteger la información. Aprenderás cómo funcionan los algoritmos que mantienen segura nuestra comunicación.

## Temario

1.  **Conceptos Básicos**: Texto claro, texto cifrado, llaves.
2.  **Tipos de Cifrado**:
    - **Simétrico**: AES, DES, ChaCha20.
    - **Asimétrico**: RSA, ElGamal, Curva Elíptica (ECC).
3.  **Funciones Hash**: MD5, SHA-1, SHA-256, SHA-3.
4.  **Firmas Digitales y Certificados**: Infraestructura de Llave Pública (PKI).
5.  **Esteganografía**: Ocultar mensajes dentro de imágenes, audio o video.
6.  **Criptoanálisis**: Ataques de fuerza bruta y ataques de diccionario.

## Laboratorio Práctico
- Cifrado y descifrado de archivos usando GPG.
- Uso de herramientas de esteganografía como `steghide`.
