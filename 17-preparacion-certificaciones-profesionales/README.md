# Módulo 17: Preparación para Certificaciones Profesionales

La ciberseguridad es una carrera de aprendizaje continuo. Este módulo te guiará sobre qué certificaciones tomar y cómo prepararte para ellas.

## Roadmap de Certificaciones

1.  **Nivel Inicial (Entry-Level)**:
    - **CompTIA Security+**: Conocimientos generales.
    - **eJPT (Junior Penetration Tester)**: El mejor punto de partida para pentesting.
    - **ISC2 Certified in Cybersecurity (CC)**: Fundamentos sólidos.

2.  **Nivel Intermedio (Professional)**:
    - **OSCP (Offensive Security Certified Professional)**: El estándar de la industria.
    - **CompTIA CySA+ (Cybersecurity Analyst)**: Enfoque en defensa y análisis.
    - **BTL1 (Blue Team Level 1)**: Para especialistas en defensa.

3.  **Nivel Avanzado / Especializado**:
    - **CISSP**: Para gestión y arquitectura.
    - **OSCE / OSEP**: Pentesting avanzado.
    - **CISM / CISA**: Auditoría y gestión.

## Consejos para los Exámenes
- Creación de un "Cheatsheet" personal.
- Uso de simuladores de exámenes.
- Importancia de los laboratorios prácticos (TryHackMe, HackTheBox).
- Gestión del tiempo durante los exámenes prácticos (24h+).
