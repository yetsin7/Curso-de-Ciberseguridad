# Módulo 02 — Redes y Protocolos (Explicado Fácil 🌐)

¿Alguna vez te has preguntado cómo llega un mensaje de WhatsApp desde tu teléfono hasta el de tu amigo al otro lado del mundo? ¡Eso es magia de redes! En este módulo vamos a entender cómo hablan las computadoras entre sí.

---

## 🧐 ¿Qué aprenderás aquí?

Vamos a desarmar el internet para ver sus piezas. Usaremos la analogía de un sistema de correos o una cebolla.

| Concepto | Analogía de la vida real |
| :--- | :--- |
| **Dirección IP** | Tu dirección postal (calle, número, ciudad). |
| **Puerto** | El número de apartamento dentro de un edificio. |
| **Protocolo** | El idioma en el que hablan (ej. español, inglés). |
| **Router** | El cartero que sabe por qué camino ir para entregar el sobre rápido. |
| **Paquete** | Una pieza pequeña de un mensaje grande (como enviar un rompecabezas en varios sobres). |

## 🧅 El Modelo OSI: Las Capas de la Cebolla
Imagina que enviar un archivo es como preparar un regalo.
1. Envuelves el regalo (Datos).
2. Le pones una etiqueta con el nombre (Capa de transporte).
3. Lo metes en una caja con la dirección (Capa de red).
4. El camión lo lleva por la carretera (Capa física).

¡Si una capa falla, el regalo no llega!
