# Módulo 16: Proyecto Final de Ciberseguridad

Integración de todos los conocimientos en un escenario de auditoría real.

#### Archivos:
- `01_enunciado_proyecto.md`: Descripción detallada del escenario y objetivos.
- `02_entorno_laboratorio.json`: Configuración de máquinas virtuales requeridas.
- `03_checklist_auditoria.txt`: Pasos a seguir durante la ejecución.
- `04_formato_entrega.md`: Cómo estructurar el reporte final.
- `05_ejemplo_reporte_ejecutivo.md`: Modelo de comunicación de resultados a gerencia.
