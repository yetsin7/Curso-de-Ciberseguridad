# Módulo 12: Informática Forense y Respuesta a Incidentes

¿Qué sucede después de un ataque? Aprenderás a recolectar evidencias y a manejar una brecha de seguridad de forma profesional.

## Temario

1.  **Proceso de Respuesta a Incidentes**: Preparación, Identificación, Contención, Erradicación, Recuperación y Lecciones Aprendidas.
2.  **Principios Forenses**: Cadena de custodia y preservación de evidencia.
3.  **Forense de Memoria RAM**: Uso de `Volatility` para extraer procesos, conexiones y contraseñas.
4.  **Forense de Disco**: Creación de imágenes de disco y análisis de sistemas de archivos (Autopsy / Sleuth Kit).
5.  **Análisis de Logs**: Correlación de eventos en Windows y Linux.
6.  **Línea de Tiempo (Timeline Analysis)**: Reconstrucción de los eventos del ataque.

## Laboratorio Práctico
- Extracción de artefactos de una imagen de memoria RAM volcada.
- Recuperación de archivos borrados de una imagen de disco USB.
