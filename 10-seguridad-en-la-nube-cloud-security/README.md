# Módulo 10: Seguridad en la Nube (Cloud Security)

La infraestructura se mueve a la nube. Aprenderás las responsabilidades y riesgos específicos de AWS, Azure y Google Cloud.

## Temario

1.  **Modelo de Responsabilidad Compartida**: Qué protege el proveedor y qué proteges tú.
2.  **IAM (Identity and Access Management)**: Gestión de identidades y permisos granulares.
3.  **Seguridad en el Almacenamiento**: Protegiendo Buckets S3 y bases de datos.
4.  **Vulnerabilidades Comunes en la Nube**:
    - Buckets mal configurados (exposición pública).
    - Fuga de llaves de API.
    - Ataques de Server-Side Request Forgery (SSRF) en instancias cloud.
5.  **Cumplimiento en la Nube**: Uso de herramientas nativas como AWS GuardDuty o Azure Sentinel.

## Laboratorio Práctico
- Configuración segura de un Bucket en AWS S3.
- Auditoría de permisos de un rol IAM usando herramientas como `Pacu`.
